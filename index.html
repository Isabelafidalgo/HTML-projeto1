História dos Computadores

Os computadores têm uma história* fascinante que remonta ao século XIX. Desde as máquinas mecânicas da primeira geração, como Harvard Mark I em 1994 até o século XXI nos dias de hoje com a quarta geração e seus avanços eletrônicos. Como o ENIAC, os computadores evoluíram para se tornar a base da era da informação.

Algumas das personalidades importantes para os avanços dessa área foram Grace Murray Hopper, Joseph Carl Robnett Licklider, Robert E. Kahn, Tim Berners-Lee, Marc Andreessen, Bill Gates e Stive Jobs.

História da Internet

A internet teve início nos anos 1960 como um projeto militar nos EUA. Ao longo das décadas, ela se expandiu globalmente, conectando bilhões de dispositivos. O surgimento da World Wide Web na década de 1990 impulsionou a internet para sua forma atual, facilitando o compartilhamento de informações em escala global.

Clients e Servers


(Voltar ao início)

Na arquitetura cliente-servidor, os clientes são dispositivos que solicitam serviços, enquanto os servidores fornecem esses serviços. É uma abordagem eficiente para distribuir recursos e processamento, com os clientes fazendo solicitações e os servidores respondendo.

Linguagens de programação

Uma linguagem de programação é um conjunto de regras e comandos que permite que um programador instrua um computador a executar tarefas específicas. Linguagens como Python, Java e C++ são exemplos comuns usados para criar software.

Começando com HTML


(Voltar ao início)

HTML significa (Hypertext Markup Language) e apesar do nome não é uma linguagem de programação e sim um arquivo de marcação padrão para criar páginas da web. Ela utiliza uma estrutura de marcação para definir a estrutura e o conteúdo de uma página, usando tags para identificar elementos como títulos, parágrafos e imagens.

  • Tags

    • Tags**são elementos fundamentais em linguagens de marcação como HTML. Elas são usadas para envolver e definir diferentes partes de conteúdo na página. Tags são geralmente compostas por uma abertura (< tag > ) e um fechamento (< /tag >).

  • Tags de Texto

    • Tags de texto em HTML servem para formatação e organização de um texto e incluem < p > para parágrafos, < h1 > a < h6 > para títulos, < mark > para marcar e destacar uma parte do texto, < i > para itálico, < u > para suplinhar, < strong > para negrito, < blockquote > para citações e muitas outras tags e marcadores para editar e foramtar os textos criados por HTML.

  • Tags de Listas ordenadas e não ordenadas

    • Para criar listas ordenadas, utiliza-se a tag < ol >, e para listas não ordenadas, a tag < ul >. Cada item da lista é definido pela tag < li >.

  • Tags de Links

    • A tag < a > em HTML é usada para criar links. O atributo "href" especifica o destino do link. Exemplo: < a href="https://exemplo.com" >Texto do Link< /a >.
